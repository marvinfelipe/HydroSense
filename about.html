WHAT IS HYDROSENSE?

HydroSense is an innovative web application that provides users with tailored suggestions for water purification methods based on real-time sensor readings. It analyzes data such as Total Dissolved Solids (TDS), turbidity, water temperature, and pH to generate specific and effective water treatment recommendations. HydroSense simplifies complex data into user-friendly guidance, offering easy, homemade treatments that require no specialized equipment.

The diagram shows the ESP32-WROOM-32D microcontroller connected to an Arduino Uno and various sensors (TDS, turbidity, temperature, pH) via interface boards. The Arduino Uno transmits data to the ESP32 via serial connection. The ESP32 also connects to a microSD card for storage and an OLED display for interfacing.

The HydroSense system shows sensors measuring pH, TDS, temperature, and turbidity in water. These sensors connect to an Arduino Uno, which sends data to an ESP32-WROOM-32D microcontroller. The microcontroller displays data on an OLED screen and logs it on a microSD card.
